## 🤖 L'Innovation : Chatbot Intelligent Intégré
WEST_PMS se distingue par son assistant IA intégré, conçu pour accompagner les ingénieurs et gestionnaires au quotidien :
- **Interaction en Langage Naturel** : Posez des questions sur l'état des projets en cours.
- **Support Automatisé** : Assistance rapide sur les différents modules de la plateforme.
- **Interface Intégrée** : Un chatbot accessible en un clic depuis le tableau de bord.

## 📋 Modules Principaux
- **Gestion de Projets d'Ingénierie** : Planification, suivi des jalons (milestones) et livrables techniques.
- **Pilotage RH** : Attribution des tâches aux ingénieurs et suivi de la charge de travail.
- **Reporting & Analytics** : Graphiques de progression et analyse de performance via Chart.js.
- **Gestion Documentaire** : Centralisation des rapports et plans techniques.

## 🛠️ Stack Technique
- **Framework** : Laravel 10 (PHP 8.2)
- **Intelligence Artificielle** : API OpenAI (Chatbot contextuel)
- **Base de données** : MySQL
- **Frontend** : Blade Engine, Bootstrap 5, JavaScript (Vite)

## ⚙️ Installation
1. `git clone https://github.com/lassana99/west-pms-gestion-de-projet.git`
2. `composer install && npm install`
3. Configurez votre fichier `.env` (Base de données et clé API OpenAI).
4. `php artisan migrate`
5. `php artisan serve`
