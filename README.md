## L'Innovation : Chatbot Intelligent Intégré
WEST_PMS intègre un assistant virtuel (Chatbot) pour accompagner les utilisateurs :
- **Requêtes Contextuelles** : Interrogez l'IA sur l'état des projets.
- **Support Rapide** : Aide à la navigation et aux procédures internes de West Ingénierie.
- **Interface PHP/JS** : Communication fluide entre le frontend et l'API OpenAI via PHP.

## Modules de Gestion
- **Gestion de Projets** : Création, suivi d'avancement et archivage des dossiers techniques.
- **Suivi d'Équipe** : Attribution des missions aux ingénieurs.
- **Reporting visuel** : Tableaux de bord avec graphiques (Chart.js) pour la performance.
- **Gestion Documentaire** : Centralisation des pièces jointes et plans.

## Stack Technique
- **Langage** : PHP Natif (Procédural/Objet)
- **Base de données** : MySQL
- **Frontend** : HTML5, CSS3 (Bootstrap 5), JavaScript
- **IA** : API OpenAI (GPT)
- **Visualisation** : Chart.js
